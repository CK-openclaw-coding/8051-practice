# 8051 刷題練習 - CH10：定時器基本動作

### 1. 在 8051 微控制器中， Timer 的主要功能為何？
- (A) 儲存資料
- (B) 計時或計數外部事件
- (C) 控制 I/O
- (D) 傳輸資料
**答案：(B)**

### 2. 8051 內建幾個定時器？
- (A) 1 個
- (B) 2 個（ Timer0 與 Timer1 ）
- (C) 3 個
- (D) 4 個
**答案：(B)**

### 3. Timer 在「 Timer 模式」下，其計數來源為何？
- (A) 外部脈波
- (B) 內部機器週期
- (C) UART
- (D) Reset
**答案：(B)**

### 4. Timer 在「 Counter 模式」下，其計數來源為何？
- (A) 內部時脈
- (B) 外部輸入腳位脈波
- (C) RAM
- (D) ROM
**答案：(B)**

### 5. Timer0 的控制暫存器為何？
- (A) SCON
- (B) TMOD
- (C) IE
- (D) PCON
**答案：(B)**

### 6. Timer 啟動控制位元為何？
- (A) TF
- (B) TR
- (C) IE
- (D) EA
**答案：(B)**

### 7. 當 Timer 溢位時， TF 旗標會如何？
- (A) 清除
- (B) 置 1
- (C) 不變
- (D) 歸零
**答案：(B)**

### 8. Timer 溢位後若未使用中斷，需如何處理 TF ？
- (A) 自動清除
- (B) 需由程式清除
- (C) Reset
- (D) 不需處理
**答案：(B)**

### 9. Timer 的計數單位在 12MHz 下為多少？
- (A) 1 μs
- (B) 2 μs
- (C) 0.5 μs
- (D) 12 μs
**答案：(A)**

### 10. 若 Timer 計數速度越快，代表什麼？
- (A) 時脈越低
- (B) 時脈越高
- (C) 記憶體增加
- (D) CPU 停止
**答案：(B)**

### 11. 在 8051 定時器運作中，若設定為 Timer 模式且系統時脈為 12 MHz ，則定時器每增加 1 計數值 所需的時間為多少？（假設 1 機器週期 = 12 個時脈週期）
- (A) 1 μs
- (B) 0.5 μs
- (C) 12 μs
- (D) 2 μs
**答案：(A)**

### 12. 在 Counter 模式下（ C/T = 1 ）， 8051 的 Timer0 計數來源為何？
- (A) 內部機器週期
- (B) 外部腳位 T0 （ P3.4 ）的輸入脈波
- (C) UART 時脈
- (D) Reset 訊號
**答案：(B)**

### 13. Timer0 的溢位旗標 TF0 設為 1 的時機為何？
- (A) Timer 啟動時
- (B) Timer 計數達到最大值並回到 0 時
- (C) Reset 時
- (D) Timer 停止時
**答案：(B)**

### 14. 若使用 Timer0 並希望在溢位時自動觸發中斷，除了啟動 TR0 外，還需設定哪一項？
- (A) 設定 EA = 1 與 ET0 = 1
- (B) 設定 PCON
- (C) 設定 SCON
- (D) 設定 ALE
**答案：(A)**

### 15. 在未使用中斷的情況下，當程式透過輪詢方式偵測 TF0 旗標，溢位後通常應進行何種操作？
- (A) 重設 CPU
- (B) 清除 TF0 並重新載入初值
- (C) 關閉電源
- (D) 改變 I/O
**答案：(B)**

### 16. Timer0 與 Timer1 在功能上最主要的差異為何？
- (A) Timer1 不能計時
- (B) Timer1 可用於串列通訊波特率產生
- (C) Timer0 無法使用
- (D) Timer1 沒有模式
**答案：(B)**

### 17. 在 8051 中， TR0 位元的主要功能為何？
- (A) 設定 Timer 模式
- (B) 啟動或停止 Timer0
- (C) 清除溢位
- (D) 控制 I/O
**答案：(B)**

### 18. 若程式設計中忘記將 TR0 設為 1 ，則 Timer0 會有什麼結果？
- (A) 正常計數
- (B) 完全不運作
- (C) 自動 Reset
- (D) 亂數計數
**答案：(B)**

### 19. 在 Timer 設計中，若初始值設定過小（接近最大值），會造成什麼結果？
- (A) 溢位時間變長
- (B) 溢位時間變短
- (C) 不影響
- (D) 無法運作
**答案：(B)**

### 20. Timer 計算延遲時間時，主要依據哪兩個因素？
- (A) RAM 與 ROM
- (B) 初始值與時脈頻率
- (C) I/O 與 UART
- (D) Reset 與電壓
**答案：(B)**
