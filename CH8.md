# 8051 刷題練習 - CH8：中斷系統

### 1. 在 8051 微控制器中，中斷（ Interrupt ）的主要功能為何？
- (A) 增加記憶體容量
- (B) 讓 CPU 在特定事件發生時暫停目前程式並轉去執行服務程式
- (C) 提高時脈頻率
- (D) 控制電源
**答案：(B)**

### 2. 8051 中斷系統中，所有中斷是否預設為啟用？
- (A) 是
- (B) 否，需要透過 IE 暫存器啟用
- (C) 只啟用 Timer
- (D) 只啟用外部中斷
**答案：(B)**

### 3. IE （ Interrupt Enable ）暫存器的主要用途為何？
- (A) 儲存中斷資料
- (B) 控制是否允許中斷發生
- (C) 控制時脈
- (D) 控制 I/O
**答案：(B)**

### 4. 在 IE 暫存器中， EA （ Enable All ）位元的功能為何？
- (A) 控制 Timer
- (B) 啟用所有中斷總開關
- (C) 控制 UART
- (D) 控制 Reset
**答案：(B)**

### 5. 若 EA = 0 ，即使個別中斷已啟用，結果為何？
- (A) 中斷仍可執行
- (B) 所有中斷均被禁止
- (C) 只有 Timer 可執行
- (D) 只有外部中斷可執行
**答案：(B)**

### 6. 8051 中的外部中斷 INT0 對應哪個腳位？
- (A) P3.0
- (B) P3.1
- (C) P3.2
- (D) P3.3
**答案：(C)**

### 7. 外部中斷 INT1 對應哪個腳位？
- (A) P3.2
- (B) P3.3
- (C) P3.4
- (D) P3.5
**答案：(B)**

### 8. 外部中斷觸發方式可設定為何種型態？
- (A) 只限電壓準位觸發
- (B) 只限邊緣觸發
- (C) 電壓準位或邊緣觸發
- (D) PWM 觸發
**答案：(C)**

### 9. TCON 暫存器中 IT0 位元的功能為何？
- (A) 控制 Timer
- (B) 設定 INT0 的觸發方式
- (C) 控制 UART
- (D) Reset
**答案：(B)**

### 10. 若 IT0 = 1 ， INT0 為何種觸發方式？
- (A) 低電位觸發
- (B) 高電位觸發
- (C) 負緣觸發
- (D) 正緣觸發
**答案：(C)**

### 11. 當中斷發生時， CPU 會先執行何種動作？
- (A) Reset
- (B) 跳至中斷向量位址
- (C) 清除 RAM
- (D) 停止運作
**答案：(B)**

### 12. 中斷服務程式（ ISR ）結束時應使用何指令返回？
- (A) RET
- (B) RETI
- (C) JMP
- (D) CALL
**答案：(B)**

### 13. RETI 與 RET 的差異為何？
- (A) 無差異
- (B) RETI 會通知中斷系統
- (C) RETI 用於 Timer
- (D) RET 用於 UART
**答案：(B)**

### 14. IP （ Interrupt Priority ）暫存器的功能為何？
- (A) 設定中斷優先權
- (B) 控制記憶體
- (C) 控制時脈
- (D) 控制 I/O
**答案：(A)**

### 15. 若兩個中斷同時發生， CPU 會如何處理？
- (A) 隨機執行
- (B) 依優先權處理
- (C) 同時執行
- (D) 忽略
**答案：(B)**

### 16. 高優先權中斷是否可中斷低優先權中斷？
- (A) 否
- (B) 是
- (C) 不一定
- (D) 無法判斷
**答案：(B)**

### 17. 低優先權中斷是否可中斷高優先權中斷？
- (A) 是
- (B) 否
- (C) 只限 Timer
- (D) 只限 UART
**答案：(B)**

### 18. 中斷巢狀（ Nested Interrupt ）發生條件為何？
- (A) 多個中斷同時啟用且有優先權區分
- (B) 只有一個中斷
- (C) 無中斷
- (D) Reset
**答案：(A)**

### 19. 外部中斷旗標 IE0 何時被清除？
- (A) 手動
- (B) ISR 執行時自動清除（邊緣觸發）
- (C) Reset
- (D) Timer
**答案：(B)**

### 20. 中斷系統的主要優點為何？
- (A) 增加記憶體
- (B) 提高即時反應能力
- (C) 降低功耗
- (D) 增加電壓
**答案：(B)**

### 21. 在 8051 中，若同時有多個相同優先權的中斷發生， CPU 將依照什麼順序決定執行哪一個中斷服務 程式？
- (A) 隨機決定
- (B) 依中斷向量位址順序（固定優先順序）
**答案：(B)**
