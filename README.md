# 8051 單晶片刷題練習題庫

這個專案收錄了 8051 單晶片微控制器的練習題，涵蓋時脈、中斷、定時器、I/O 通訊及鍵盤掃描等章節。

## 內容章節
- CH7：時脈與機器週期
- CH8：中斷系統
- CH9：I/O 資料轉移方式
- CH10：定時器基本動作
- CH11：4×4 鍵盤掃描
- CH12：串列 I/O 通訊
