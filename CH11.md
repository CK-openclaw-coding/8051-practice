# 8051 刷題練習 - CH11：4×4 鍵盤掃描

### 1. 4×4 鍵盤矩陣共有幾個按鍵？
- (A) 4
- (B) 8
- (C) 16
- (D) 32
**答案：(C)**

### 2. 4×4 鍵盤通常需要幾條 I/O 線？
- (A) 4
- (B) 8 （ 4 列＋ 4 行）
- (C) 16
- (D) 2
**答案：(B)**

### 3. 鍵盤矩陣設計的主要目的為何？
- (A) 增加速度
- (B) 減少 I/O 腳位使用
- (C) 增加電壓
- (D) 增加記憶體
**答案：(B)**

### 4. 在掃描鍵盤時，通常會採用什麼方式？
- (A) 全部輸入
- (B) 列掃描或行掃描
- (C) DMA
- (D) Reset
**答案：(B)**

### 5. 鍵盤掃描的基本原理為何？
- (A) 同時讀取所有鍵
- (B) 逐行或逐列輸出，讀取另一方向
- (C) 使用 UART
- (D) 使用 Timer
**答案：(B)**

### 6. 若某一列設為低電位，其他列為高電位，按鍵按下時會發生什麼？
- (A) 無變化
- (B) 對應行會變為低電位
- (C) 全部變高
- (D) Reset
**答案：(B)**

### 7. 鍵盤未按下時，輸入端通常為何狀態？
- (A) 低電位
- (B) 高電位（上拉）
- (C) 浮動
- (D) 無訊號
**答案：(B)**

### 8. 為避免按鍵誤判，需加入什麼機制？
- (A) Timer
- (B) 去彈跳（ Debounce ）
- (C) Reset
- (D) DMA
**答案：(B)**

### 9. 去彈跳通常透過什麼方式實現？
- (A) 高電壓
- (B) 延遲或軟體判斷
- (C) 增加記憶體
- (D) UART
**答案：(B)**

### 10. 在鍵盤掃描程式中，最常使用哪種 I/O 方式？
- (A) DMA
- (B) Polling
- (C) Reset
- (D) ADC
**答案：(B)**

### 11. 在 4×4 鍵盤掃描中，若所有列（ Columns ）皆設為高電位，而行（ Rows ）設為輸入，當按鍵未被按 下時，輸入端通常呈現何種狀態？
- (A) 低電位
- (B) 高電位（由上拉電阻維持）
- (C) 浮動
- (D) 隨機
**答案：(B)**

### 12. 在掃描過程中，若某一列被設為低電位，並偵測到某一行變為低電位，代表什麼？
- (A) 無按鍵按下
- (B) 該列與該行交叉的按鍵被按下
- (C) 電路故障
- (D) Reset
**答案：(B)**

### 13. 鍵盤掃描程式中，通常會逐列輸出低電位，其主要目的為何？
- (A) 減少電壓
- (B) 確定按鍵位置
- (C) 提高速度
- (D) 控制 Timer
**答案：(B)**

### 14. 若掃描速度過慢，可能會造成什麼問題？
- (A) CPU 過載
- (B) 按鍵反應延遲
- (C) 記憶體不足
- (D) Reset
**答案：(B)**

### 15. 若未加入去彈跳（ Debounce ）處理，可能導致什麼現象？
- (A) 按鍵無效
- (B) 一次按壓被判定為多次輸入
- (C) CPU 停止
- (D) 電壓過高
**答案：(B)**

### 16. 去彈跳時間通常設定為多少範圍較合理？
- (A) 1 ns
- (B) 約 5 ～ 20 ms
- (C) 1 秒
- (D) 1 分鐘
**答案：(B)**

### 17. 若同時按下多個鍵，鍵盤矩陣可能產生什麼問題？
- (A) 無法輸入
- (B) 鬼鍵（ Ghost Key ）現象
- (C) Timer 錯誤
- (D) Reset
**答案：(B)**

### 18. 為避免鬼鍵問題，可採用何種方法？
- (A) 提高電壓
- (B) 加入二極體隔離
- (C) 增加 RAM
- (D) 使用 DMA
**答案：(B)**

### 19. 在 鍵盤掃描中，通常哪一方向設為輸出？
- (A) 行
- (B) 列
- (C) 任意
- (D) RAM
**答案：(B)**

### 20. 鍵盤掃描最常搭配哪種 I/O 方式？
- (A) DMA
- (B) Polling
- (C) Reset
- (D) ADC
**答案：(B)**
