# 8051 刷題練習 - CH12：串列 I/O 通訊

### 1. 串列通訊（ Serial Communication ）的主要特性為何？
- (A) 同時傳送多位元資料
- (B) 一次只傳送 1 位元資料
- (C) 不需時脈
- (D) 無需傳輸線
**答案：(B)**

### 2. 串列通訊相較於並列通訊，其最大優點為何？
- (A) 傳輸速度一定較快
- (B) 使用較少傳輸線
- (C) 不需控制電路
- (D) 無需同步
**答案：(B)**

### 3. 下列何者屬於串列通訊常見應用？
- (A) LED 矩陣
- (B) UART 通訊
- (C) RAM 存取
- (D) DMA 控制
**答案：(B)**

### 4. 同步串列通訊（ Synchronous Communication ）最大的特點為何？
- (A) 不需時脈訊號
- (B) 傳輸雙方共用時脈
- (C) 一次傳送 8 位元
- (D) 不需同步
**答案：(B)**

### 5. 非同步串列通訊（ Asynchronous Communication ）中，用來表示資料開始的訊號稱為？
- (A) Stop Bit
- (B) Start Bit
- (C) Parity Bit
- (D) Clock Bit
**答案：(B)**

### 6. 在非同步串列通訊中， Stop Bit 的主要功能為何？
- (A) 表示資料開始
- (B) 表示資料結束
- (C) 提高速度
- (D) 增加電流
**答案：(B)**

### 7. 波特率（ Baud Rate ）的定義為何？
- (A) 電壓大小
- (B) 每秒傳輸的符號數
- (C) 記憶體容量
- (D) CPU 速度
**答案：(B)**

### 8. 若串列通訊雙方波特率設定不同，可能造成什麼問題？
- (A) 傳輸更快
- (B) 資料錯誤或無法通訊
- (C) 記憶體增加
- (D) Timer 停止
**答案：(B)**

### 9. Parity Bit （同位元檢查位元）的主要用途為何？
- (A) 控制時脈
- (B) 錯誤檢查
- (C) 提高電壓
- (D) 控制 I/O
**答案：(B)**

### 10. 串列通訊中最常見的資料格式為何？
- (A) 5-bit
- (B) 8-bit 資料格式
- (C) 32-bit
- (D) 64-bit
**答案：(B)**

### 11. EIA-232 標準最常見名稱為何？
- (A) USB
- (B) RS-232
- (C) SPI
- (D) I2C
**答案：(B)**

### 12. RS-232 主要用途為何？
- (A) 高速影像
- (B) 電腦與周邊設備串列通訊
- (C) 記憶體控制
- (D) DMA
**答案：(B)**

### 13. RS-232 的訊號電壓範圍通常為？
- (A) 0V ～ 5V
- (B) ±3V ～ ±15V
- (C) 0V ～ 1V
- (D) 12V 固定
**答案：(B)**

### 14. RS-232 中，邏輯 1 通常代表何種電壓？
- (A) 正電壓
- (B) 負電壓
- (C) 0V
- (D) 5V
**答案：(B)**

### 15. RS-232 中，邏輯 0 通常代表何種電壓？
- (A) 負電壓
- (B) 正電壓
- (C) 0V
- (D) 浮接
**答案：(B)**

### 16. 8051 TTL 電平若要與 RS-232 通訊，通常需搭配什麼 IC ？
- (A) 7447
- (B) MAX232
- (C) ULN2003
- (D) ADC0804
**答案：(B)**

### 17. MAX232 的主要功能為何？
- (A) 放大電流
- (B) TTL 與 RS-232 電平轉換
- (C) 控制 Timer
- (D) 顯示解碼
**答案：(B)**

### 18. RS-232 的主要缺點為何？
- (A) 無法通訊
- (B) 傳輸距離與速度有限
- (C) 電壓不足
- (D) 不需接地
**答案：(B)**

### 19. RS-232 通常使用哪種接頭？
- (A) USB
- (B) DB9 或 DB25
- (C) HDMI
- (D) RJ45
**答案：(B)**

### 20. RS-232 屬於何種類型通訊？
- (A) 並列
- (B) 點對點串列通訊
- (C) 無線
- (D) 光纖
**答案：(B)**
