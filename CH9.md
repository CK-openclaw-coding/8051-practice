# 8051 刷題練習 - CH9：I/O 資料轉移方式

### 1. 在微控制器系統中，所謂 I/O 資料轉移方式，主要是指 CPU 與周邊裝置之間資料交換的控制方 式。以下何者不屬於常見的 I/O 資料轉移方式？
- (A) 輪呼式（ Polling ）
- (B) 中斷式（ Interrupt-driven ）
- (C) 直接記憶體存取（ DMA ）
- (D) 編譯式（ Compile-driven ）
**答案：(D)**

### 2. 在輪呼式（ Polling ） I/O 中， CPU 會反覆檢查某一裝置狀態旗標，其主要缺點為何？
- (A) 電路複雜
- (B) CPU 時間被浪費在等待
- (C) 無法讀資料
- (D) 記憶體不足
**答案：(B)**

### 3. 中斷式 I/O 的最大優點為何？
- (A) 設計簡單
- (B) CPU 不需持續等待裝置
- (C) 無需控制訊號
- (D) 不需程式
**答案：(B)**

### 4. DMA （ Direct Memory Access ）的主要特性為何？
- (A) CPU 負責所有資料搬移
- (B) 資料由周邊直接與記憶體交換
- (C) 只用於 UART
- (D) 速度最慢
**答案：(B)**

### 5. 在 DMA 模式下， CPU 的角色通常為何？
- (A) 完全停止
- (B) 僅初始化與控制
- (C) 直接搬移資料
- (D) 控制 I/O 埠
**答案：(B)**

### 6. 輪呼式 I/O 最適合用於哪種情況？
- (A) 高速資料傳輸
- (B) 低速或簡單裝置
- (C) 大型系統
- (D) DMA 系統
**答案：(B)**

### 7. 中斷式 I/O 與輪呼式相比，其主要改善為何？
- (A) 記憶體增加
- (B) CPU 利用率提升
- (C) 電壓增加
- (D) I/O 數量增加
**答案：(B)**

### 8. DMA 傳輸完成後，通常會透過何種方式通知 CPU ？
- (A) Reset
- (B) 中斷
- (C) 輪詢
- (D) 無通知
**答案：(B)**

### 9. 在高速資料傳輸應用中，最適合使用哪種方式？
- (A) 輪呼式
- (B) 中斷式
- (C) DMA
- (D) 手動
**答案：(C)**

### 10. 若系統設計需即時回應但資料量不大，最適合的方式為？
- (A) DMA
- (B) 中斷式
- (C) 輪呼式
- (D) Reset
**答案：(B)**

### 11. 在嵌入式系統中，當設計者選擇 I/O 資料轉移方式時，若希望在不大量占用 CPU 資源的情況下處 理周邊裝置事件，應優先考慮哪一種方式？
- (A) 輪呼式（ Polling ）
- (B) 中斷式（ Interrupt-driven ）
- (C) 忙碌等待（ Busy Waiting ）
- (D) 手動控制
**答案：(B)**

### 12. 若系統需要處理大量且連續的資料傳輸（例如影像或高速感測資料），為了減少 CPU 負擔並提高 效率，最適合採用哪種 I/O 資料轉移方式？
- (A) 輪呼式
- (B) 中斷式
- (C) DMA （直接記憶體存取）
- (D) 軟體延遲
**答案：(C)**

### 13. 在比較三種 I/O 資料轉移方式（ Polling 、中斷、 DMA ）時，下列何者關於 CPU 使用率的敘述最正 確？
- (A) Polling 使用率最低
- (B) DMA 使用率最高
- (C) DMA 使用率最低
- (D) 三者相同
**答案：(C)**

### 14. 在系統設計中，若裝置回應時間不固定且需要即時處理，但資料量不大，應優先選擇哪種方式？
- (A) Polling
- (B) DMA
- (C) 中斷式 I/O
- (D) Reset
**答案：(C)**

### 15. 下列關於 DMA 的敘述，何者最正確？
- (A) CPU 必須逐筆搬移資料
- (B) DMA 控制器可直接在記憶體與周邊之間傳輸資料
- (C) DMA 僅適用於低速系統
- (D) DMA 無需控制訊號
**答案：(B)**

### 16. 在輪呼式 I/O 中， CPU 會透過何種方式判斷裝置是否準備好？
- (A) 中斷
- (B) 讀取狀態旗標
- (C) Reset
- (D) Timer
**答案：(B)**

### 17. 輪呼式 I/O 的基本流程為何？
- (A) 中斷 → 處理
- (B) 檢查 → 等待 → 處理
- (C) DMA→ 處理
- (D) Reset→ 執行
**答案：(B)**

### 18. 若裝置長時間未準備好， Polling 系統會如何？
- (A) CPU 執行其他任務
- (B) CPU 持續等待
- (C) 自動 Reset
- (D) 停止
**答案：(B)**

### 19. Polling 最主要的效率問題來自於？
- (A) 記憶體
- (B) 空轉等待（ Busy Waiting ）
- (C) 電壓
- (D) Timer
**答案：(B)**

### 20. Polling 的優點為何？
- (A) 高效率
- (B) 程式簡單
- (C) 無需 CPU
- (D) 無需記憶體
**答案：(B)**
