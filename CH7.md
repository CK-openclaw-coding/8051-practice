# 8051 刷題練習 - CH7：時脈與機器週期

### 1. 在標準 8051 微控制器中，一個機器週期（ Machine Cycle ）包含多少個時脈週期（ Clock Cycles ）？
- (A) 6
- (B) 12
- (C) 24
- (D) 48
**答案：(B)**

### 2. 若系統晶振為 12 MHz ，則一個機器週期時間為多少？
- (A) 1 μs
- (B) 2 μs
- (C) 0.5 μs
- (D) 12 μs
**答案：(A)**

### 3. 某系統使用 6 MHz 晶振，請問一個機器週期的時間為何？
- (A) 1 μs
- (B) 2 μs
- (C) 3 μs
- (D) 0.5 μs
**答案：(B)**

### 4. 在 8051 中，大部分指令的執行時間是以何種單位表示？
- (A) 秒
- (B) 電壓
- (C) 機器週期
- (D) 位元數
**答案：(C)**

### 5. 若某指令需要 2 個機器週期，在 12 MHz 下執行時間為多少？
- (A) 1 μs
- (B) 2 μs
- (C) 3 μs
- (D) 4 μs
**答案：(B)**

### 6. 提高晶振頻率時，對系統效能的影響為何？
- (A) 降低速度
- (B) 提高執行速度
- (C) 無影響
- (D) 增加記憶體
**答案：(B)**

### 7. 時脈週期（ Clock Cycle ）的定義為何？
- (A) 一條指令時間
- (B) 振盪器完成一次震盪所需時間
- (C) 記憶體讀取時間
- (D) 中斷時間
**答案：(B)**

### 8. 機器週期與時脈週期的關係為何？
- (A) 相同
- (B) 機器週期由多個時脈週期構成
- (C) 無關
- (D) 相反
**答案：(B)**

### 9. 若使用 11.0592 MHz 晶振，其最常見用途為何？
- (A) 降低功耗
- (B) 精確產生串列通訊 Baud Rate
- (C) 增加 RAM
- (D) 控制 LED
**答案：(B)**

### 10. 在高速系統中，時脈頻率過高可能導致什麼問題？
- (A) 記憶體增加
- (B) 功耗與發熱增加
- (C) 指令變少
- (D) UART 停止
**答案：(B)**

### 11. 在標準 8051 架構中，若某一指令需執行 3 個機器週期，且系統使用 12 MHz 晶振，請問該指令 的執行時間約為多少？
- (A) 1 μs
- (B) 2 μs
- (C) 3 μs
- (D) 4 μs
**答案：(C)**

### 12. 若系統晶振頻率改為 24 MHz （仍為 12T 架構），則與 12 MHz 相比，系統整體執行速度會如何 變化？
- (A) 不變
- (B) 增加為 2 倍
- (C) 降低為一半
- (D) 隨機變化
**答案：(B)**

### 13. 在 8051 中，機器週期的長短會直接影響哪一項系統性能？
- (A) RAM 容量
- (B) 指令執行速度
- (C) I/O 腳數量
- (D) 中斷數量
**答案：(B)**

### 14. 若使用 11.0592 MHz 晶振，選擇此頻率的主要原因為何？
- (A) 降低功耗
- (B) 提供較高運算速度
- (C) 方便產生標準串列通訊速率（ Baud Rate ）
- (D) 增加記憶體容量
**答案：(C)**

### 15. 在 Timer 設計中，若時脈頻率越高，則 Timer 計數的解析度會如何變化？
- (A) 降低
- (B) 提高
- (C) 不變
- (D) 無法判斷
**答案：(B)**

### 16. 若某系統使用低頻晶振（例如 1 MHz ），則最可能出現的系統特性為何？
- (A) 高速運算
- (B) 低功耗但速度較慢
- (C) 記憶體增加
- (D) UART 速度變快
**答案：(B)**

### 17. 在設計延遲程式時，若未考慮機器週期，最可能造成什麼問題？
- (A) 記憶體溢位
- (B) 延遲時間不準確
- (C) CPU 停止
- (D) I/O 錯誤
**答案：(B)**

### 18. 若一個 Timer 設計依據 12 MHz ，但實際晶振為 11 MHz ，將導致什麼結果？
- (A) 計時更準
- (B) 計時偏慢
- (C) 計時偏快
- (D) 無影響
**答案：(B)**

### 19. 機器週期主要由哪個硬體決定？
- (A) RAM
- (B) 晶振頻率
- (C) I/O
- (D) UART
**答案：(B)**

### 20. 在高精度應用（如通訊或測量）中，最重要的時脈特性為何？
- (A) 高電壓
- (B) 穩定與準確頻率
- (C) 大容量
- (D) 多 I/O
**答案：(B)**

### 21. 在 8051 外部記憶體系統中， ALE （ Address Latch Enable ）訊號的主要功能為何？
- (A) 控制中斷
- (B) 鎖存低位址（ A0–A7 ）
- (C) 控制資料傳輸
- (D) Reset 系統
**答案：(B)**

### 22. PSEN （ Program Store Enable ）訊號主要用於何種操作？
- (A) 寫入外部 RAM
- (B) 讀取外部記憶體
- (C) 控制 Timer
- (D) 控制 I/O
**答案：(B)**

### 23. PSEN 訊號屬於哪種類型？
- (A) Active High
- (B) Active Low
- (C) 雙向
- (D) 類比
**答案：(B)**

### 24. ALE 訊號在系統中通常與哪個元件搭配使用？
- (A) RAM
- (B) 鎖存器（ Latch ）
- (C) UART
- (D) Timer
**答案：(B)**

### 25. 若 ALE 訊號失效，系統最可能出現什麼問題？
- (A) Timer 停止
- (B) 位址鎖存錯誤
- (C) UART 錯誤
- (D) LED 不亮
**答案：(B)**
